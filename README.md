# Sudoku-Generator
 By Jerry Zhu

This Sudoku Generator is a Java program I created that generates a random, valid 9x9 Sudoku board and prints it to the console. It works by first filling the board with a shift pattern that satisfies all Sudoku rules, then shuffling rows and columns within their bands and reassigning all digit values to randomize the output. To run the program, compile and run SudokuGenerator.java in a website such as JavaSandbox. The project includes the code file and the design document.
